# Joyeux Anniversaire Mon Ange Loren

## Site Web d'Anniversaire Romantique

Un site web exceptionnel créé avec amour pour célébrer l'anniversaire de Loren.

## Comment Utiliser

### Pour Ouvrir le Site Localement :
1. Double-cliquez sur `index.html`
2. Ou faites un clic droit -> "Ouvrir avec" -> Votre navigateur web

### Pour Partager le Site :
**Méthode 1 - Lien Direct :**
- Copiez tous les fichiers du dossier `Special`
- Uploadez-les sur un service d'hébergement gratuit (GitHub Pages, Netlify, etc.)
- Partagez le lien généré

**Méthode 2 - Fichier ZIP :**
- Compressez tout le dossier `Special` en ZIP
- Envoyez le ZIP par email/messagerie
- La personne dézippe et ouvre `index.html`

## Fichiers Principaux

- `index.html` - Structure principale du site
- `style.css` - Styles et animations
- `script.js` - Interactions et effets spéciaux
- Dossier images/vidéos - Vos souvenirs personnalisés

## Fonctionnalités Spéciales

- **Modal d'agrandissement** : Cliquez sur n'importe quelle photo/vidéo pour l'agrandir
- **Feux d'artifice interactifs** : Bouton "Clique ici" dans la lettre
- **Animations romantiques** : Coeurs flottants, particules, effets visuels
- **Suivi des visites** : Savoir quand Loren visite le site
- **Design responsive** : Fonctionne sur tous les appareils

## Messages Personnalisés

Chaque photo et vidéo contient des messages romantiques uniques qui racontent votre histoire d'amour.
